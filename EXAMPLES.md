# Примеры конвертации файлов

## Excel файлы → CSV

### Пример 1: Один лист
**Исходный файл:** `отчет_2024.xlsx`
**Лист:** "Данные"

**Результат:**
```
отчет_2024.xlsx.csv
```

---

### Пример 2: Несколько листов
**Исходный файл:** `продажи.xlsx`
**Листы:** "Январь", "Февраль", "Март"

**Результат:**
```
продажи.xlsx_лист1_Январь.csv
продажи.xlsx_лист2_Февраль.csv
продажи.xlsx_лист3_Март.csv
```

---

### Пример 3: Название листа с запрещенными символами
**Исходный файл:** `budget.xlsx`
**Листы:** "Q1/2024", "Q2:2024", "Summary*Final"

**Результат:**
```
budget.xlsx_лист1_Q1_2024.csv      (/ заменен на _)
budget.xlsx_лист2_Q2_2024.csv      (: заменен на _)
budget.xlsx_лист3_Summary_Final.csv (* заменен на _)
```

**Запрещенные символы:** `< > : " / \ | ? *` - все заменяются на `_`

---

### Пример 4: Длинное название листа
**Исходный файл:** `data.xlsx`
**Лист:** "Очень длинное название листа которое превышает допустимый лимит в 50 символов"

**Результат:**
```
data.xlsx_лист1_Очень_длинное_название_листа_которое_превыш.csv
```
(Обрезано до 50 символов)

---

### Пример 5: Пустое название листа
**Исходный файл:** `test.xlsx`
**Листы:** "", "Sheet1", "   "

**Результат:**
```
test.xlsx_лист1_Лист.csv       (пустое название заменено)
test.xlsx_лист2_Sheet1.csv
test.xlsx_лист3_Лист.csv       (пробелы заменены)
```

---

## Parquet файлы → CSV

**Исходный файл:** `sales_data.parquet`

**Результат:**
```
sales_data.parquet.csv
```

---

## Word документы → Markdown (без изображений)

**Исходный файл:** `document.docx` (содержит текст и 3 изображения)

**Результат:** `document.docx.md`
```markdown
---
source_file: document.docx
original_format: .docx
converted_by: WordConverter
---

# document

Это текст документа.

[Изображение удалено]

Еще текст.

[Изображение удалено]

Заключение.

[Изображение удалено]
```

---

## Пропускаемые файлы

### Видео файлы
```
video.mp4    → НЕ загружается
meeting.webm → НЕ загружается
demo.avi     → НЕ загружается
```

### Изображения
```
logo.png      → НЕ загружается
photo.jpg     → НЕ загружается
diagram.svg   → НЕ загружается
screenshot.bmp → НЕ загружается
```

## Структура папки после синхронизации

**До синхронизации (Яндекс.Диск):**
```
/Клиенты
├── Компания_А
│   ├── отчет.xlsx (3 листа)
│   ├── презентация.pptx
│   ├── договор.docx (с изображениями)
│   └── видео_встречи.mp4
├── Компания_Б
│   ├── данные.parquet
│   ├── лого.png
│   └── скриншот.jpg
```

**После синхронизации (локально):**
```
localdata/
├── downloaded_files/
│   ├── Компания_А
│   │   ├── отчет.xlsx (временно, удален после конвертации)
│   │   ├── презентация.pptx
│   │   └── договор.docx (временно, удален после конвертации)
│   └── Компания_Б
│       └── данные.parquet (временно, удален после конвертации)
│
└── markdown_files/
    ├── Компания_А
    │   ├── отчет.xlsx_лист1_Январь.csv
    │   ├── отчет.xlsx_лист2_Февраль.csv
    │   ├── отчет.xlsx_лист3_Март.csv
    │   ├── презентация.pptx.md
    │   └── договор.docx.md (без изображений)
    └── Компания_Б
        └── данные.parquet.csv
```

**Пропущено:**
- `видео_встречи.mp4` - не загружено
- `лого.png` - не загружено
- `скриншот.jpg` - не загружено
